# 🚀 Crypto Monitor 加密貨幣監控系統

自動監控 BTC/ETH，透過 Discord Webhook 發送通知。

## 📊 監控功能

### 價格監控
| 幣種 | 危險線 | 壓力位 | 支撐位 |
|------|--------|--------|--------|
| BTC | $59,800 / $52,000 | $67,500 / $68,800 / $71,500 / $72,000 / $74,000 / $76,000 | $66,000 / $65,700 / $63,800 |
| ETH | $1,750 / $1,500 | $1,980 / $2,000 / $2,100 / $2,150 / $2,250 | $1,920 / $1,900 / $1,850 |

### RSI 監控 (15m / 30m / 1H / 4H)
- 🔴 RSI < 30 → 超賣警報
- 🟢 RSI > 70 → 超買警報

### OB 訂單塊 (15m / 30m)
- 識別看漲/看跌訂單塊
- 價格接近 OB 區域時通知

### 其他監控
- OI 變動 > 1.5%
- 價格波動 > 1% (5分鐘內)

## 🔧 API 來源
優先順序：Bybit → OKX → CoinGecko

## ⚙️ 設定方式

1. Fork 這個 Repo
2. Settings → Secrets → Actions
3. 新增 `DISCORD_WEBHOOK_URL`
4. Actions 會自動每 15 分鐘執行

## 📋 執行頻率

| 任務 | 頻率 |
|------|------|
| 價格/OI/RSI | 每 15 分鐘 |
| OB 訂單塊 | 每 15 分鐘 |
| 倉位顧問 | 每小時 |
